Variables

Introduction

变量与寄存器不同。
在Ex模式，或VimL中，直接使用
let var_name=value 来保存值到变量。
变量的内容可以是：

  1. 数字 -- let a=1
  2. 字符串 -- let b='hello'
  3. 数组(学名叫List） -- let c=[1,2,'3']
  4. 字典 -- let d={ 'hello':'world', 'Yadli' : 'Katana'}。使用[]访问，例如 echo d['hello']

将变量输出到Buffer：pu=var_name，注意'='表示"Expression"。详见:help pu
此外，形如@[a-zA-Z0-9!@#($"]（符号是我乱打的。。。不要参考！）的表达式将引用 寄存器的值。

Scope

形如l:var_name的是局部变量。在当前Buffer有效。
形如s:var_name的是Script变量，在当前VimL Script中有效。
形如g:var_name的是全局变量，你懂的。
