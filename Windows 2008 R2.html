Windows 2008 R2

一激动，把Win7给格式化掉了，费了九牛二虎之力（丢人吗？）装上了2008R2SP1。
Very impressive！价值几千美元的东西就是不一样……解压900MB文件，速度竟然达到了246MB/S，其缓存性能有多变态，可想而知。
再搭配上DreamSpark上搞下来的VS2010专业版，恩，活生生一套完整的干活工具啊。

安装途中其实还是遇到不少的问题，有大有小，现在列一下：

  • 安装方法，自然是用HDD Installer扒皮再直接硬盘装了！
  • 安装到最后（重启一次），提示"Windows couldn't configure to run on this computer hardware"。
    • 尝试了很多种方法，更是把安装盘换成了SP1，仍然未果。后经谢老板指点，拆了一块硬盘，单硬盘毫无压力地装好了。
  • http://www.win2008r2workstation.com/ 这里有一个很有意思的网站，专门教你怎么用一个昂贵的服务器OS来打酱油 :-) :-) :-)
  • 一定要关闭IE ESC（Enhanced Security Control），完全是坑爹，神马都不能访问/下载。
  • Wireless Lan Service需要自己安装，以获取WIFI机能
  • 手动启动Audio服务以获得声音机能
  • 显卡驱动，直接用Driver Genius探测找到WIN7 X64驱动即可。
  • 安装Desktop Experience机能，开启Themes服务，获得Aero机能

基本上上述步骤搞定之后，这个系统的易用性就和普通家用WIN7差也不多了。但是人家性能在那摆着，完全具有压倒性的优势。
游戏功能我还没有测试。但是说实话，当一群人可以玩PlayStation的时候，谁没事玩电脑游戏呢？又不热闹。。。

神马？Windows Search不见了？我曾经如此讨厌这个功能，现在却又迫切地需要。。。
Okay，它藏在Server Management -> Add Role -> File Services里面。
